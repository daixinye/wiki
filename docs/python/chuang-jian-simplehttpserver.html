创建SimpleHTTPServer

创建SimpleHTTPServer

在终端中切换到包含HTML文件的目录。

python -m SimpleHTTPServer & open http://0.0.0.0:8000/index.html

即可启动一个HTTP服务器，并打开目录中的主页。
